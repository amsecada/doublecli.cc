# Site Design Rules

This document outlines the core aesthetic and functional design rules for `doublecli.cc`.

## Global Aesthetics
- **Section Kickers**: Every section eyebrow (kicker) must include a leading styled dot (e.g., `<span className="badge-dot" />`) to maintain visual consistency across all pages.
- **Premium Vibe**: Focus on high contrast, dynamic animations, and low-footprint designs to build trust and convey an expensive, hacker arcade aesthetic.

## Navigation
- **Logo Link**: The main brand logo (`doublecli.cc`) in the header must always link back to the homepage (`/`).

*Keep this document updated as new overarching aesthetic decisions or global layout rules are established.*
